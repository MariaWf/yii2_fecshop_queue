Fecshop 队列扩展


> Fecshop queue 功能部分，用于扩展一些队列的功能

1.安装
--------

```
composer require --prefer-dist fancyecommerce/fecshop_queue 
```

or 在根目录的`composer.json`中添加

```
"fancyecommerce/fecshop_queue": "~1.0.3"

```

然后执行

```
composer update
```
